# Academic Cover Page Generator
A professional, web-based tool designed for university students to generate standardized Lab Reports and Assignment cover pages instantly.

What is it?
This is a lightweight, "What You See Is Get" (WYSIWYG) document creator built with HTML, CSS, and JavaScript. It allows students to input their course details, student information, and teacher's data to produce a perfectly formatted A4 cover page that meets academic standards.

How it works:
Dynamic Input: Users fill out a sidebar with details like University Name, Course Code, Student ID, and Batch.

Live Preview: The document updates in real-time on the right side of the screen as you type.

Smart Formatting:

Lab Report Mode: Displays "Experiment No" and "Date of Experiment."

Assignment Mode: Switches labels to "Assignment Topic" and automatically hides the experiment date for a cleaner look.

Logo Customization: Users can upload their own university logo directly from their computer to personalize the document.

Print-Optimized: The tool uses specific CSS @media print rules to ensure that when you "Export to PDF," the sidebar and web buttons disappear, leaving only a clean, professional 1-inch margin document without browser headers or footers.

Tech Stack:
HTML5 & CSS3: For the structural layout and print-perfect styling.

JavaScript (Vanilla): For real-time data binding and image processing.
